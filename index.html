Как добавить новый сайт

  1. Загрузите новый HTML-файл в корень репозитория (через интерфейс GitHub или git push).
  2. Отредактируйте этот файл index.html — добавьте карточку в сетку по образцу выше.
  3. Не забудьте увеличить счётчик в блоке Всего сайтов.
  4. Сохраните изменения (commit). Через минуту новый сайт появится на главной.

Для файла с пробелами используйте в ссылке %20 вместо пробела, либо переименуйте файл (рекомендуется). Например: Sans%20Serif%20Fonts.html.
